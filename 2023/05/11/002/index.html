准备工作：

注册gitee账号

注册gitee账号

创建一个仓库，命名为blog

选择私有（仅仓库成员可见）

勾选初始化仓库、设置模版，并勾选Readme文件、选择分支类型

安装软件

Node.js

需安装12.0及以上版本，官网下载LTS版本即可。

请确保已勾选Add to PATH选项（默认勾选）

Git

下载Standalone Installer的64位版本即可。

Typora

VS Code

下载个可编辑MD文件与HTML文件的编程软件，这里我推荐这两个软件。

写文章用Typora，部署用VS Code。

打开我们建好的仓库，点击管理-部署公钥管理，添加公钥

标题输入你的邮箱，然后粘贴你的公钥。

出现 shell access即代表成功。

若出现Permission denied (publickey).，则重新打开记事本，刷新一下文件夹，重新打开文件夹复制。

博客部署：

tip：每次修改完配置后，都要保存，然后执行以上命令。

clean 命令会导致壁纸和头像文件被删除，请做好备份。

现在，恭喜你，你已经成功建好你的博客了。

设置链接：

打开你所创建的仓库，点击服务。

选择Gitee Pages，勾选强制使用HTTPS，点击启动。

自动生成的网址，就是你的博客地址。

需要注意，每次部署完成，需要去Gitee Pages更新一下地址。
