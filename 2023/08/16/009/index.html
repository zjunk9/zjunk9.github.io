国际商务考点汇总

发表于2023-08-16|更新于2023-08-16

生产基本要素：土地，劳动力，资本，企业

第一产业（原材料）：自然资源的开发

第二产业（加工）：制造业和建筑业

第三产业（服务）：服务业

初级市场–第一产业市场：未经加工的初级产品体量巨大，价格也因此相对便宜。

二级市场–第二产业市场：二级市场中，生产者除了生产本国需要的产品外，还要生产其他国家需要的产品。

三级市场–第三产业市场：三级市场中的服务需要一定程度的变化，技术服务不需要改变或改变很少。

专业化（劳动分工）：是使人们能够提供其最擅长的产品和服务的一种生产方式。

专业化的优点：

  1. 每个员工都能做他们最擅长的工作。
  2. 减少对工具的需求成本
  3. 可以购买专业的机器设备
  4. 无需转换流程，节约时间
  5. 花费更少的时间掌握专门的技能
  6. 雇主能够更准确的估计生产成本和产量。

内部规模经济：技术经济、财务经济、管理经济、商业经济、风险承担经济、研发经济。

技术经济：使企业实现劳动分工

财务经济：使企业获得更低的融资利率

管理经济：聘请专门人才，有助于实现更多的管理经济

商业经济：企业大批量采购，可以获得较大的折扣，降低生产成本

风险承担经济：更多了解业内行情，能够有效的规避风险和降低成本

研发经济：可以设计更具有竞争力的新产品或者新的生产流程

外部规模经济：劳动力集中培训学习求职带动的教育培训、支柱产业需要的支持性的辅助产业、支持性服务产业、产业集聚带来的机会和声誉影响。

绝对优势理论（亚当·斯密提）：是指在同样的数量资源的情况下，一个生产商可以生产出比另一个生产商更多的特定产品

比较优势理论（大卫·李嘉图）：当一个生产商在生产某件特定的商品时比另一个生产商效率相对更高。

机会成本：是生产某种特定商品的成本，它用我们不能生产的另一种商品来衡量。

贸易制裁的因素：

  1. 约束国际贸易的战略性因素：个别政府或许在制定国内生产产品的决策时并不与国际贸易理论相一致。
  2. 约束国际贸易的政治因素：有时国家彼此结成联盟，当然这些联盟没有必要一定是正式的。联盟国之间建立了亲密关系，而有些国家却永远不能成为同盟。

战略分析方法包括：

  1. SWOT分析：是只指用于分析企业与竞争者相比自身所具备的内部优势(S)和劣势(W)，并识别出外部的主要机会(O)与威胁(T)。

优势：受到良好激励的员工、优质的产品、良好的声誉、现代化的生产技术。

劣势：员工士气低落和动机不足、陈旧的设备、糟糕的质量控制、落后的信息管理系统、研究与开发进度缓慢、不良的资金状况。

机会：利用现有的资源开拓新市场、新立法或新趋势为开发新产品或服务提供的机会、技术进步为改善制造技术或管理系统提供的机会

威胁：竞争激烈程度增加、不断改进的技术、经济因素、新立法、抵制变革的老员工、管理者的自满、组织缺乏制定决策和实施控制所需要的资金或其他信息。

  1. PEST分析：是指考虑可以影响企业活动的外部因素，并分析政治因素(P)、环境因素(E)、社会因素(S)和技术因素(T)。

政治因素：政治稳定性、政治体制、政府经济政策、政府经济计划、立法

经济因素：失业率、经济增长率、通货膨胀、税收、利率、汇率。

社会因素：生活方式及闲暇时间、文化、人口统计特征、时尚、教育、压力群体、环境问题。

技术因素：新产品和服务、通信、信息技术、新制造流程、新材料、消费者期望、更新速度。

  1. 波特五力分析：波特五力分析的关键因素是进入者的威胁、替代品的威胁、供应商和客户的议价能力、行业内现有企业之间的竞争程度。

新产品开发的过程：创意的产生–创意筛选–商业分析–产品开发–市场测试–投放市场

产品政策和战略需要考虑的因素

标准化产品或修改产品：营销人员需要考虑产品在市场期间应该保持何种程度的标准化。

产品生命周期：导入期–成长期–成熟期–衰退期

产品组合分析–波士顿矩阵。波士顿矩阵根据市场相对份额和市场增长率对产品进行分类。

  1. 明星类产品：高增长率高份额，处于产品生命周期早期阶段。企业在广告和促销上花费大量金钱。
  2. 现金牛产品：低增长率高份额，处于产品生命周期成熟阶段。具有良好的声誉，无需大量促销。
  3. 瘦狗类产品：低增长率低份额，处于产品生命周期衰退阶段。不再投资广告和促销，应该退出市场。
  4. 问题类产品：高增长率低份额，需要大量促销来增加市场份额和树立形象。

产品和服务策略中要考虑的因素：

  1. 产品的性质：无形性、易逝性、异质性、不可分割性。
  2. 产品的利益：性能、形象、感知价值。
  3. 属性：品牌名称、产品设计、产品包装、产品标签。
  4. 营销支持：交货、保证、售后服务、安装。

国际市场调研方法：

  1. 跨文化调研
  2. 多国调研
  3. 扫描国际市场
  4. 营销情报
  5. 职能调研

市场调研的过程：

  1. 界定问题
  2. 选择方法
  3. 调研设计
  4. 实施调研
  5. 分析数据
  6. 报告和展示

国外市场营销的原因：

  1. 规模经济
  2. 市场拓展
  3. 新市场的机遇
  4. 竞争
  5. 风险分散
  6. 经济条件
  7. 技术因素

国际市场营销环境：

  1. 社会和文化：物质文化、语言因素、宗教信仰、教育、社会组织。
  2. 法律环境：法律渊源：习惯法、普通法、伊斯兰法律、犹太法。国际法律：国际公法、国际私法或冲突法、超国家法（欧盟法）。
  3. 政治环境：运营限制、歧视性限制、直接抵制行为、国际关系、稳定性。
  4. 经济：发达国家、新兴经济体、欠发达国家的国际营销。
  5. 技术环境：国际标准书号、因特网、互联网、电子信息高速公路、技术/文化悖论

政府干预带来的问题：

  1. 政府干预可以保护衰退产业和新兴产业
  2. 阻止倾销
  3. 可以保护国内产业
  4. 有助于缓解失业

政府调控出口的措施：

  1. 出口补贴。政府可以通过向国内一些企业发放补贴规避冲突，从而鼓励这些企业出口产品。
  2. 出口管制（降低出口的措施）。政府通过出口配额或者对出口商品征收出口税的措施来降低出口。

政府减少进口的措施：

  1. 提高进口商品或服务的关税。
  2. 设定产品的进口配额
  3. 外汇管制
  4. 实物管制

国际收支平衡表：用于记录一国与世界上其他国家之间的所有外部交易。国际收支=经常项目+资本项目

国际收支平衡表的借方总额和贷方总额是相等的，其差额为零。

经常项目：记录了一国进口或出口的所有产品与服务的贷币价值。经常项目分为有形贸易收支和无形贸易收支。

有形贸易收支差额（通常所讲的贸易差额）=出口商品总额-进口商品总额

无形贸易收支差额=出口服务总额-进口服务总额

经常项目=有形贸易收支差额+无形贸易收支差额

贸易条件：衡量一国之间进出口价格变化的一种方法。

贸易条件指数=单位出口价值/单位进口价值x100=出口价格指数/进口价格指数x100

汇率：是一种价格，即以一种货币的另一种货币的价格。

汇率制度的变革过程：

易货制（一种支付国外商品和服务的方式）–金本位制（一种参照一定量黄金决定汇率的固定汇率体制）–布雷顿森林体系的固定汇率制度（是指利率不是由市场决定的，而是由政府决定的）–浮动汇率制度（利率完全由需求和供给力量决定，不是受政府的干预，货币没有保证的价值）

硬通货：如果一种货币价值不断突破其上限，他就被称为“硬通货”

货币投机：是指在一段时间内使用少量的其他货币购买大量的特定货币的个人。

牛市：货币的价格走势可能是上升的。

热钱：如果某个国家的利率上升，而其他国家所有国家利率不变，那么投机者就有可能通过向高利率货币转移而获得好处。

常用的展示统计数据的方法：

  1. 数据表格
  2. 曲线图
  3. 饼状图和柱状图
  4. 诸如象形图和统计图等图表。

二手数据：包括现有的公共出版的材料，由其他组织收集的。包括：

  1. 内部资料
  2. 政府公布数据
  3. 数据库
  4. 在线数据资源、电子商务和因特网营销
  5. 万维网

一手数据调研：二手数据可能在发布时已经过时，或者不能提供公司真正需要的详细信息，所以很多公司开始收集一手资料。

各种调查方法比较：

  1. 面对面调查：可以解散迷惑，但会产生偏差
  2. 电话调查：价格便宜、易于管理、短时间容易得到大量的被调查者，但受时间的限制、被调查者不愿配合、给出错误答案
  3. 邮寄调查：便宜，但回复很少
  4. 网络调查：成本低，不受时间限制，但被调查者范围受限制。

企业可以采用四种扩大产品销量或市场退出战略（安索夫矩阵）：

巩固：在现有市场营销现有产品；市场渗透建议利用现有产品对现有市场进行进一步的渗透。

产品开发：是针对现有市场开发新产品。

市场开发：是为现有产品寻找新的市场，如果更好的锁定顾客、市场调研和市场细分。

多元化：多元化会改变企业核心业务，可能涉及将企业现有的生产与相关业务整合（新产品新市场）。

安索夫市场矩阵：

  1. 市场渗透战略：是通过挖掘现有产品的潜力获取较大的市场份额。
  2. 产品发展战略：企业在现有市场引进新产品，主要包括产品开发，产品延伸。
  3. 市场发展战略：将现有产品推出新市场
  4. 多元化战略：是企业将业务拓展到新产品和新市场。进一步划分为：水平一体化、垂直一体化、集中战略和集团战略。

波特的一般战略：

  1. 总成本领先战略：要求企业在保持产品质量的同时生产成本比竞争对手要低。
  2. 差异化战略：以创造竞争者无法效仿的独特性为基础，能够为顾客提供独特的价值。
  3. 目标聚焦战略：企业选择特定的细分市场作为竞争基础，而不是整个行业。

定价的影响因素：

  1. 市场因素：消费者是否愿意接受该价格
  2. 产品因素：是定价的基础，目标获得可接受的利润。
  3. 地方性因素：产品会因为出口增加税收而使成本增加
  4. 成本：国际产品要考虑运输成本和税收成本。

定价目标：

  1. 投资回报率：根据预期的销售量来测算价格
  2. 市场稳定：新进入市场的企业除非实力非常强大，应采取与竞争者对手接近的价格进入，以保持市场的稳定。
  3. 抵制新进入者：降低价格。
  4. 市场渗透：定价低。
  5. 产品差异化：定价高。

线下推广方法：

  1. 确保公共关系：确保展现企业的良好形象的重要途径。
  2. 直接邮寄：在发达地区是最直接的方法，但是回报率低。
  3. 信息保护法：助于目标国信息立法。
  4. 国际销售促销：贸易促销和消费者促销。

分销：是将商品从供应商手中运到顾客手中的物理过程，并且涉及到储存、运输和装卸。

出口的方式：直接出口和间接出口

直接出口：由产品的生产商亲自操作出口业务，而不是委托他人。

间接出口：企业的产品在国外市场销售，而企业并不参与国际营销活动，由代理商和经销商自行安排分销渠道。

间接出口方法：

  1. 外国买主订购产品供自己使用或在自己国家的市场上转售。
  2. 利用专门从事出口业务的出口代理机构；适合那些无法承担出口业务的中小企业。
  3. 背驮式出口（附带式出口）：指一家生产商凭借强大的海外分销系统出口自己生产的产品的同时还附带出口其他企业的产品。这种生产商被称为出口载体，通常是拥有完整的出口设施和分销渠道的大企业。

国际市场人员需要关注的风险：

  1. 市场风险：消费者可能不再购买产品或停止购买。
  2. 金融风险：资本的变化；汇率变化不利，通货膨胀发生，债务违约，目标国家设置关税
  3. 法律风险：产品责任、目标国家的贸易壁垒、合同违约、代理问题
  4. 监管风险：目标国家改变了环境法、目标国家改变了对外贸易规则。

企业目标：

  1. 利润最大化
  2. 高的市场份额
  3. 销售收入最大化
  4. 高增长率
  5. 在更广泛的市场上运营
  6. 保证稳定的收入
  7. 顾客满意度等

反馈：通过对比实际取得的结果和预期结果，并以系统取得的成果为基础对体系进行修改。反馈是计划与控制的联系环节。

积极反馈：系统偏离计划方向的程度在持续甚至有所增加，这样的反馈通常是有利的。

消极反馈：当结果显示系统已经背离了计划方向的时候，消极反馈出现了。

可控因素：决策人员拥有一定自行决定权的可投入因素。如轮班因素

不可控因素：属于外部或者环境因素，决策人员在短期内无法控制。如原材料成本增加，税收增加。

贸易集团的优势：

  1. 创立后的集团比各个成员国自身的经济实力都要强，因而能够与更强的一些单个国家抗衡。
  2. 创立后的集团内各成员国之间的自由贸易给彼此带来了利益，各成员国都获得更大的市场以及由于企业产出增长而带来的规模经济。
  3. 区域内单个国家可能无力支持发展某一特定行业，但是如果各个国家携手合作就会造就特定产业，这将促进区域一体化的发展。
  4. 贸易增长会带来更多的财富以及更高的生活质量，而更大程度上的自给自足将改善各成员国的收支平衡。

自由贸易区：是指成员国之间同意在其内部商品自由流动。这意味着国家都不会强制征税或利用配额来限制其他成员国的商品。各成员国分别执行与非成员国的收支优势。

关税同盟：是指成员国之间没有关税，对外部非成员国采用统一或共同的关税政策，强化了各成员国政府之间的整合，避免了自由贸易区的行政管理问题。

共同市场：超越了关税同盟，旨在协调、规范有关贸易的各项规章，在成员国之间商品、劳务和资产都可以自由流动，成员国政府之间相互协调方针政策。

经济同盟：是共同市场的更高阶段，要求成员国所有经济政策协调一致。只有一个央行，只有一种货币，并且有共同的财务政策和货币政策。

政治同盟：如果国家之间的政治理念相似，或者说他们长期合作的传统，即使不是邻国，也可以形成联盟或达成协议。

区域贸易领域的特点：

  1. 成员国之间的距离很近。
  2. 消费者品位可能很相似，有利于在邻近国家建立分销渠道。
  3. 邻近国家可能有共同的历史和利益，很容易协调其政策。

欧盟的组织机构：

  1. 欧洲委员会
  2. 欧洲部长理事会
  3. 欧洲议会
  4. 欧洲法院

物理壁垒：欧盟取消了所有对边境货物流动的管制，取消了对人口流动的海关管制。

技术性贸易壁垒：欧盟成员国对大部分产品已经达成了相互认可各国规定的原则。

税收壁垒：部分国家销售税的调整减少了税收壁垒。

公法壁垒：不管是国家地区还是地方机构招标的公共采购合同都面向欧盟的任何投标人。
