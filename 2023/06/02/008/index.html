从零开始的windows系统指南2.0

发表于2023-06-02|更新于2024-08-21

作者有话要说：

距离第一次写windows系统指南已经过去3年之久（2020年11月3日开始撰写），期间经过15次版本更新，最近一次更新还是在2022年12月1日，距离这次撰写已经快要半年之久。作者从石墨文档开始创作，后来石墨加了广告影响使用，转到了语雀，现在语雀收费搞心态，免费用户不能使用暗黑模式和互联网公开，于是又转到了金山文档这里，希望金山文档能更加好用吧！

前言：

本文一开始的目的是为了自己今后装机方便，后发现他人也有此方面的需求，故此分享出来，作者有强迫症，看不得别人电脑广告弹窗满天飞。文章出现的软件大多数都是作者使用过感觉良好的，考虑到有些软件官方下载对网络有所需求，因此这里为方便大家，提供了我自用的版本。自用版本受时效限制，因此作者会不定时更新，但肯定不如官方更新快。自用版本皆经作者测试，放心可用。文章分为4个部分：系统篇、装机篇、推荐篇、优化篇。大家可以按需浏览。

本次windows系统指南2.0全新升级：

  1. 采用☆☆☆☆☆推荐指数（“无”基本为必备），让大家更好选择自己所需的软件
  2. 为方便大家下载，自用软件（尽量下自用版的）分享皆采用不限速的网盘分享
  3. 汇总网盘链接（大部分自用都在，密码在公众号里回复“密码”）
  4. 看不习惯排版的，可前往文档版（已停更）

系统篇

本篇分为微软官方系统和第三方优化系统。图方便可以使用第三方系统，保留基础功能，并且预装了输入法，浏览器等必备功能。官方系统是纯原生微软系统，需要配合激活软件使用。

官方原版

HelloWindows

推荐指数：☆☆☆☆☆
一个托管与Github的精校 完整 极致 windows系统下载仓储站，内含Windows XP-11的系统，还有Server、ARM、LTSC/LTSB等版本下载，网站本身无任何广告。除此之外，还有下载工具和激活工具供大家使用。

MSDN

推荐指数：☆☆☆☆☆
提供BT下载和百度网盘下载，涵盖系统版本Window XP-11的系统，都是原版的系统。有自带系统安装器。

ITELLYOU

推荐指数：☆☆☆☆
老牌系统下载网站，扣一星是因为要登录，旧版网站不用登录，但是现在已经不更新了（旧版网站）。不用注册，支持多平台登录。可下载Windows、Linux、macOS、Chromium OS、BSD等系统。

第三方优化版

Windsys Project

推荐指数：☆☆☆☆☆
也许是最适合年轻人使用的第三方社区系统，采用全自动部署的系统，系统纯净无广告。基于原版作出了一定的修改，使安装更方便、同时进行了一些个性化设置，但用起来的感觉几乎一致。

推荐指数：☆☆☆☆
高人气官改作者，官网需众筹才能下载，这里贴一些免费下载的网址。

小鱼儿YR

推荐指数：☆☆☆☆
作者开电脑店的，非纯净系统，含有大量实用软件，基本安装可直接用，适用于系统封装。

和不忘初心一样，是要付费才能下载的，这里贴一些免费的网站。

本地安装

微PE

推荐指数：☆☆☆☆☆
这个PE可以部署在本地，在电脑上直接使用，非常方便，还很简洁，纯净没有广告。

一键ghost

吾爱/硬盘版/CGI/OneKey

推荐指数：☆☆☆☆
远古装机软件，程序员装机都用这个。网上有很多个版本，随便下一个就可以用，这里列出几个好用的。支持系统格式：GHO、WIM、SWM（支持格式各版本有所不同）

WinToHDD

推荐指数：☆☆☆
这个下载方式有点繁琐，只放了自用版，想要下官方直接百度就行。软件可以重装系统，克隆系统，本地安装建议安装使用，支持系统格式：ISO，WIM，ESD，SWM，VHD，VHDX

U盘(PE)安装

Fir PE

推荐指数：☆☆☆☆☆
这款pe，占用空间不大，界面美观，无广告。喜欢他的美化，真的是年轻人的第一款PE（误）。
2023.01.01：FirPE目前已更新本地安装，习惯用这个PE的可以用这个，不然本地还是推荐微PE，这个安装包有点大。

Edgeless

推荐指数：☆☆☆☆☆
这款pe，相交于上面，可以扩展应用，甚至可以作为系统日用，同样无广告。

优启通/U盘魔术师USM/微PE

推荐指数：☆☆☆☆
优启通：采用双pe模式，内存小
USM：对老机型兼容好，可以作为系统日用，内存占用较大
微PE：曾经最好的pe，现在万年不更新

DDU（显卡驱动程序卸载工具）

官网/自用版

推荐指数：无
Display Driver Uninstaller（简称 DDU）是一款显卡驱动程序删除工具，可以帮助您从系统中彻底卸载 AMD/NVIDIA/INTEL 显卡驱动程序、程序包和剩余项（包括注册表项、文件夹和文件、驱动程序存储等）。
有时候自动下载的驱动可能会不兼容，这时我们就要卸载去品牌官网下载新版驱动安装，在这之前就需要使用这个软件来卸载驱动，除此之外，它还可以阻止系统更新时更新驱动。官网下载较慢，这里提供自用版。

解压工具

WinRAR

自用旧版/自用新版/官网

推荐指数：☆☆☆☆
这款软件不用多说，很强，xp时代就已经很好用了，建议使用自用版本，烈火汉化。

7-ZIP

官网/自用

推荐指数：☆☆☆☆☆
7-Zip是一款开源免费并号称有着现今最高压缩比的压缩文件管理器，它不仅支持独有的7z文件格式，而且还支持各种其它压缩文件格式，其中包括ZIP, RAR, CAB, GZIP, BZIP2和TAR等等。此压缩软件的压缩比要比普通ZIP文件高30-50%，因此，它可以把Zip格式的文件再压缩2-10%。

NanaZip

微软商店/官方

推荐指数：☆☆☆☆☆
可能是Win11里最好的解压软件，适配Window11右键，基于开源软件7-ZIP制作。

Bandizip

官网/自用旧版/高木同学

推荐指数：☆☆☆☆
Bandizip 是一款可靠和快速的压缩软件，它支持 WinZip、7-Zip 和 WinRAR 以及其它压缩格式。它拥有非常快速的压缩和解压缩的算法，适用于多核心压缩、快速拖放、高速压缩等功能。自用版不能升级，安装完成要去设置关闭自动更新。
建议下载高木同学的版本。高木同学的说明：
第一个 zip 是 64 位官方安装包+破解文件
第二个 zip 是官方安装包( 64 32 arm 均可安装使用) + 64 位和 32 位的破解文件
第三个 zip 是 64 位和 32 位的破解文件
第四个 zip 是绿色版，解压即可使用
第五个 exe 是直装破解版
官方便携版不支持添加右键，且禁用了修复压缩包和密码恢复功能

360压缩

国内官网/国际官网

推荐指数：☆☆☆☆

浏览器

国内：

360极速浏览器X

官网/自用/极简版

推荐指数：☆☆☆☆☆
国内内核最新的64位浏览器，自己设置一下搜索引擎什么的，基本没有广告。算是我用了最久的浏览器了，一开始用的是360极速浏览器，后面官方推出X，又换到X。官网有较多版本，推荐X和极简版，极简版官网已经隐藏，这里放个下载链接。
自用版集成了flash，官方已经不支持flash，需要flash的也可以下载360安全浏览器。
自用版内核更新方法：
方法一（推荐）：下载好安装程序，用7-z点击解压，将里面的版本号文件夹和360chromex.exe，移动替换到360ChromeX\Chrome\Application中，一定不可以使用原本的exe文件，不然无法识别的。
方法二（不推荐）：删除或移走360ChromeX\Chrome\Application\360chromex.exe，然后点击安装程序，选择你一起安装的文件夹，一般为360ChromeX，直接覆盖安装，这样扩展那些都可以保存下来，我推荐使用第一种方法，因为失败了还可以退回原版本，不至于丢失数据。
还有，为了避免每次都要重建桌面快捷方式，你可以向我一样，复制360chromex.exe到360ChromeX\Chrome，为快捷方式，然后在发送到桌面快捷方式中，这样可以以逸待劳。理论下面那个浏览器也适用。
